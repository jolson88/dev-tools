# Spacemacs Cheatsheet

## File/Buffer Management

Opening a file
```
C-x C-f
```

Switch buffers (current window)
```
C-x b
```

## Window Management

Switching to next window
```
C-x o
```

Switch to a numbered window
```
<SPC> <number-of-window>
```

## Renaming Files

Renaming files via dired mode
```
C-x, C-j
R (type in name)
C-x, k <RET>
```
